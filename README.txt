#[[#]]# ${demo1}

Visualisierung des Dijkstra-Algorithmus in Java

#[[##]]# Installation

1. Installieren Sie zunächst java fx über diesen Link (https://openjfx.io/openjfx-docs/)
2. Nach der Installation öffnen Sie ein neues Java-FX-Projekt in Ihrem entsprechenden IDE
3. Importieren Sie .jar-Dateien in den Main Ordner neben der HelloApplication und dem HelloController, die automatisch von Java Fx erstellt werden

#[[##]]# Wie man das Programm ausführt

1. Das Programm sollte von der GraphVisualizer-Klasse aus ausführbar sein. Es sind keine weiteren Änderungen erforderlich, um das Programm auszuführen
